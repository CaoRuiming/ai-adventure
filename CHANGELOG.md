# Changelog

All notable changes to Local Adventure Engine are documented here.

## 0.1.0 — 2026-07-17

- Released the local terminal engine with validated authored worlds, deterministic event state, SQLite session history, lore retrieval, LM Studio structured-output support, undo, branches, checkpoints, and privacy-conscious audits.
- Added Markdown and JSON session exports that exclude prompts and raw model responses.
- Added authoring, architecture, data-model, privacy, and troubleshooting documentation.
